# CLAUDE.md

Use `AGENTS.md` in this directory as the project-specific instruction
file for `phd-public`.

In brief: this is the public Czech-facing bookdown site for sharing
Juda Kaleta's PhD research with Czech history teachers, teacher
educators, and scholars. Edit source `.Rmd` files, not generated HTML;
write Czech public prose in a clear, modest, teacher-facing voice; and
verify source changes with `make build` when feasible.

Always read `status.md` and `project_status.md` before starting work in
this project, then update them after meaningful changes.
